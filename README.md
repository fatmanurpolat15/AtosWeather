## 📖 Overview

AtosWeather is a modern weather dashboard that displays real-time weather information for multiple cities through a clean and responsive user interface.

The project integrates **Open-Meteo API** with **Netlify Serverless Functions** to provide current weather conditions and short-term forecasts while demonstrating a cloud-based serverless architecture.

---

## ✨ Features

- 🌍 Search and add multiple cities
- 🌡️ Real-time weather information
- 📅 4-day weather forecast
- ☁️ Serverless API architecture with Netlify Functions
- 📱 Responsive and modern user interface
- 🔄 Automatic weather updates
- 🌍 Location-based weather search
- ⚡ API response caching for improved performance

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/fatmanurpolat15/AtosWeather.git
```

Move into the project directory

```bash
cd AtosWeather
```

Install Netlify CLI (optional)

```bash
npm install -g netlify-cli
```

Run locally

```bash
netlify dev
```
